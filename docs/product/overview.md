# Clear Dawn v0 プロダクト概要

## コンセプト

> 夜明け前の静けさの中で、今日やるべきことを決める。

Clear Dawn は、単なるタスク管理アプリではない。仕事・夢・身体・生活を同時に抱える人が、
毎日、自分の優先順位を取り戻すためのセルフマネジメントアプリである。
思考と行動を整理し、感情だけでなく仕組みで目標達成を支える。

## v0 の位置づけ

- v0 は最終的に **全機能入りの実運用アプリ** とする（プロトタイプで捨てる前提にしない）
- 最初に実装する核は **TOP Matrix**（Phase 1 仕様書・設計書 v0.3 を詳細仕様の正とする）
- v0 は将来 v1 へ **Export API 経由でデータ移行する** 前提で設計する
- UI は **UI 仕様書 v2** をデザインの正とする（[design-system.md](../design/design-system.md) に移植済み）

## v0 の完成スコープ

## 基本方針

### プロダクト

- TOP Matrix はユーザーごとに 1 つだけ。日別・週別・月別管理はしない
- TOP に履歴は持たない。履歴はメモ・振り返り・実行履歴の側に残す
- 「今日のフォーカスカード」は置かない。シート自体が思考整理と行動決定の中心

### 技術・運用

- スタック: Laravel + Inertia.js + Vue 3 + TypeScript + Tailwind CSS v4
- PC ファースト（16:9 基準で設計、実装はレスポンシブ）
- 新規ドメインテーブルの主キーは ULID。既存 `users` は BIGINT を維持（[ADR-0001](../adr/0001-ulid-primary-keys.md)）
- 全ユーザーデータは `user_id` でスコープ（[ADR-0002](../adr/0002-user-scoped-single-user-domain.md)）
- 運用環境は最初は dev / prod のみ。test 環境は後で追加
- 本番: Laravel Cloud。監視（Sentry）は後日導入（本ドキュメント時点では未着手）

### デザイン

- 外側は美しく、内側は読みやすく
- 背景は白〜生成りの大理石、サイドバーは深紫〜藤色の夜明け前グラデーション
- 月・星・植物線画・装飾ラインは補助素材
- UI は画像一枚に頼らず HTML/CSS/Vue で構成する（[ADR-0004](../adr/0004-ui-built-with-html-css-vue.md)）

## v0 に含めないもの

- ガントチャート
- 組織・店舗前提のマルチテナント機能（`Admin` / `Shop` / `tenant_id` 等の概念は持ち込まない）
- スマホ PWA の本格対応（後続フェーズで検討）
- ダークモード（推奨: v0 非対応。未決定事項として管理）
